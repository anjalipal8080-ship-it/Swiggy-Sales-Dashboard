# Swiggy Sales Analysis Dashboard 🚀

A comprehensive Data Analytics project focusing on Swiggy's sales, delivery performance, and user behavior using advanced Excel/Power BI and data analysis techniques.

## 📊 Project Overview
This project analyzes transactional data from Swiggy to uncover key business insights, understand sales distribution across cities, evaluate delivery performance, and optimize revenue tracking.

## 💡 Key Insights & Features
- **Top 5 Cities Analysis:** Identified the highest-revenue-generating cities to target localized marketing.
- **Daily Trends:** Analyzed order volume variations across weekdays and weekends.
- **Delivery Efficiency:** Tracked average delivery times and ratings to improve customer satisfaction.
- **Interactive Visualizations:** Dynamically filters data by location, date ranges, and food categories.

## 🛠️ Tech Stack & Tools
- **Data Source:** `swiggy.csv` (Included in repository)
- **Tools Used:** Excel / Power BI (Advanced Data Modeling, DAX, and Data Cleaning)
